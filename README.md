```markdown
# وام فرزند آوری

## Project Overview
وام فرزند آوری یک برنامه وبی است که به خانواده‌ها در ایران کمک می‌کند تا در راستای فرزندآوری تسهیلات مالی دریافت کنند. این برنامه شامل یک صفحه اصلی برای معرفی وام، فرم ثبت نام آنلاین برای درخواست وام و پیامی برای موفقیت در ثبت نام می‌باشد.

## Installation
برای استفاده از این پروژه، می‌توانید فایل‌های HTML را به سادگی دانلود کرده و در مرورگر خود باز کنید. هیچ وابستگی خاصی برای نصب وجود ندارد.

1. همه فایل‌های زیر را دانلود کنید:
   - `index.html`
   - `registration.html`
   - `success.html`
2. فایل `index.html` را در مرورگر خود باز کنید.

## Usage
۱. صفحه اصلی (index.html) را باز کنید تا اطلاعات مربوط به وام فرزند آوری را مشاهده کنید.  
۲. برای ثبت نام، روی دکمه "ثبت نام اکنون" کلیک کنید تا به صفحه ثبت نام (registration.html) هدایت شوید.  
۳. فرم را پر کرده و روی دکمه "ثبت درخواست" کلیک کنید.  
۴. پس از ارسال فرم، در صورت موفقیت، به صفحه موفقیت منتقل خواهید شد (success.html).  

**توجه:** عملکرد صفحه موفقیت در کد فراهم نشده است.

## Features
- **صفحه اصلی جذاب:** نمایش اطلاعات کلی درباره وام فرزند آوری.
- **فرم ثبت نام آنلاین:** شامل قسمت‌های اطلاعات شخصی، اطلاعات فرزند، اطلاعات تماس و اطلاعات وام.
- **ارائه وام با شرایط آسان:** تسهیلات مالی با شرایط پرداخت منعطف.
- **فرم اعتبارسنجی:** بررسی صحت داده‌های وارد شده در فرم.

## Dependencies
این پروژه برای استایل‌دهی به صفحه‌ها از **Tailwind CSS** و **Font Awesome** برای نمایش آیکون‌ها استفاده می‌کند. همچنین از **Google Fonts** برای استفاده از فونت فارسی Vazirmatn بهره می‌برد. این کتابخانه‌ها به طور مستقیم از CDN بارگذاری می‌شوند و نیازی به نصب محلی ندارند.

## Project Structure
```

├── registration.html     # فرم ثبت نام وام
└── success.html          # صفحه موفقیت (فعلاً خالی)
